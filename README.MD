# Frontend
This app implements frontend of Music meta data management.

### Installation and Running

 - `git clone https://github.com/sohaibfarooqi/frontend.git`.
 - `npm install`.
 - `npm start`.

To run test use

  - `npm run test`.

### Tech Stack

 - ReactJS 16
 - Redux
 - Saga
 - Webpack
 - Babel
 - Nodejs
 - ESLint
